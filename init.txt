# INIT.TXT - Application Gestion des Adhérents - Lyon Palme
# Date: 28 Janvier 2026
# Framework: Laravel 12.0 / PHP 8.4 / Tailwind CSS 4.0 / PostgreSQL (prévu) / MariaDB (actuel)

## 📋 CONTEXTE DU PROJET

### À propos du Club Lyon Palme
- **Type**: Club sportif de nage avec palmes (monopalme/bi-palmes)
- **Affiliation**: FFESSM (Fédération Française d'Études et de Sports Sous-Marins) - Comité Régional AURA
- **Localisation**: Saint-Fons / Vénissieux (piscine d'entraînement: Centre Nautique de Vénissieux)
- **Membres**: ~40 adhérents
- **Entraînements**: Lundi 19h45-21h45 (25m), Mercredi 19h45-21h45 (25m), Jeudi 20h45-21h45 (50m)

### Périmètre du Projet - 1er VOLET UNIQUEMENT
**CE PROJET CONCERNE UNIQUEMENT :** L'**Application de gestion des adhérents** utilisée par le(s) secrétaire(s) du club.

Le projet est scindé en deux volets selon le CDC :
- **1er volet (NOTRE PÉRIMÈTRE)** : Application "Inscription des adhérents" - utilisée par le secrétaire
- **2ème volet (HORS PÉRIMÈTRE)** : Site informationnel - utilisé par les nageurs (trombinoscope, annuaire, modification compte)

### Objectifs de notre application
1. **Connexion secrétaire** à l'application
2. **Visualiser les adhérents inscrits**
3. **Procéder à l'inscription des nouveaux adhérents**
4. **Archiver les anciens adhérents** (réactivation possible)

### Date de livraison finale
**27 Avril 2025** (déployé sur VPS étudiant)

## 📝 RÈGLES DE GESTION IMPORTANTES

### Inscription Adhérents
- Documents requis: fiche inscription, certificat médical, autorisation parentale (mineurs), photo
- Certificat médical valable 3 ans
- Si certificat > 1 an: questionnaire santé obligatoire
- Login initial: initialeprenom + nom
- Mot de passe initial: date de naissance (à changer 1ère connexion)

### Rôles Adhérents
- Tous sont "Nageurs" par défaut
- Rôles cumulables: Président adjoint + Responsable matériel + Entraîneur
- Rôles bureau: Président, Vice-président, Secrétaire, Secrétaire adjoint, Trésorier
- Rôles techniques: Coach (MEF1), Entraîneur (Initiateur), Responsable planning, Responsable matériel, Responsable communication

### Planning Entraînements
- 2 entraîneurs obligatoires lundi/mercredi
- 1 entraîneur jeudi
- Planning construit 2x/an (sept-déc, janv-juin)
- Permutations possibles via commentaires
- Validation par responsable planning

### Sorties Milieu Naturel
- Organisées par entraîneurs uniquement
- Infos: jour, heure RDV, mise à eau, lieu, plage, niveau
- Mention constante: "bonnet couleur + bouée signalisation obligatoire"
- Mentions hiver: "cagoule, gants, chaussons obligatoires"
- Groupes niveau créés par organisateur (hors application)

### Compétitions
- Distances variables: 1000m, 3000m, 5000m, etc.
- Modalités: bi-palme, monopalme, support, relai
- Inscription: modalité + distance + transport + hébergement
- Date limite inscription (réservations + e-nap FFESSM)

### Matériel
- Stock: monopalmes (pointures), tubas (taille unique), combinaisons (tailles), lunettes
- Code unique par matériel
- Prêt nouveaux membres
- Traçabilité: qui, quand emprunté, quand rendu
- Réparations mineures par responsable
